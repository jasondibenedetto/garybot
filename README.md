# garybot [![NPM version][npm-image]][npm-url] [![Build Status][travis-image]][travis-url] [![Dependency Status][daviddm-image]][daviddm-url]

## Installation

Clone this repository, and run:
```sh
$ yarn
```

## Usage

```js
$ yarn run bot
```
